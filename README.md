# HizKoridoru

Cross plattform implementation (Android and IOS) of the idea from an existing App
(e.g https://play.google.com/store/apps/details?id=net.baybar.otobanhizkoridoru&hl=en_US) prototypically with Xamarin.

# Why an existing App?

Simply to step into app development with Xamarin.Forms with an existing idea.

# Different features

* EntityFramework Core
* Route saving
* Usage of CollectionView for saved routes
* Long pressed event for multiple selection, bookmarking and deleting
